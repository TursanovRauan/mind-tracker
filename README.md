Современное веб-приложение, объединяющее ключевые инструменты тайм-менеджмента в одном минималистичном и удобном интерфейсе. Помогает структурировать задачи, держать фокус на главном и анализировать свою эффективность.

## ✨ Основные функции

* **Главная страница:** Приветственная панель с динамическими часами и кратким экскурсом в базовые принципы продуктивности (Помодоро, Матрица Эйзенхауэра, Правило 80/20).
* **Планировщик задач:** Интерактивный таск-менеджер с возможностью фильтрации по категориям (Работа/Личное) и приоритету (Низкий/Средний/Высокий).
* **Калькулятор продуктивности:** Утилита для расчета индивидуального КПД на основе соотношения часов работы и отдыха.
* **Таймер Pomodoro:** Кастомизируемый таймер для работы короткими сессиями с возможностью ручной настройки интервалов.
* **Полезные ресурсы:** Кураторская подборка карточек с культовыми методиками (GTD, Атомные привычки, Deep Work).
* **Форма обратной связи:** Форма контактов со стилизованной кнопкой отправки.

## 🛠 Стек технологий

* **HTML5:** Семантическая структура страниц.
* **CSS3 (Flexbox/Grid):** Современная адаптивная верстка, кастомные градиенты, размытие (backdrop-filter) и темная тема.
* **Pure JavaScript:** Логика переключения вкладок (SPA-поведение), работа таймера, расчеты калькулятора и динамическое добавление задач.
